# Aevita Clinic — Skopje

Static marketing site for **Aevita Clinic Skopje** — a premium blood-filtration and IV-wellness clinic targeting Southeastern Europe and the DACH source markets.

**Live:** https://aleksandar-hqdm.github.io/aevita-clinic/

## What's here

- **Umbrella service** (homepage): the clinic itself
- **4 core services**
  - EBO2 Blood Filtration (signature procedure)
  - MAH Ozone Therapy
  - IV Therapy Lounge (hub)
  - Medical Weight Loss — Wegovy / Ozempic / Mounjaro
- **8 IV drip sub-pages** under IV Therapy: NAD+, Myers', Glutathione, Immune (high-dose C), Energy & B-complex, Beauty & Collagen, Athletic Recovery, Hangover
- **Location matrix** — Country → City → Service
  - 7 Balkan countries: North Macedonia (HQ), Kosovo, Albania, Serbia, Bulgaria, Montenegro, Bosnia & Herzegovina
  - 22 cities total (3-4 per country)
  - Every city × every core service = a dedicated page
- **Source-market landing pages** for Switzerland, Austria, Germany
- About, Pricing, Consultation form, Contact, 404, sitemap, robots

**Total: 140 HTML pages.**

## SEO

- Schema.org markup on every page: `MedicalClinic`, `MedicalProcedure`, `BreadcrumbList`, `FAQPage`
- Canonical URLs and Open Graph tags
- `sitemap.xml` lists all 140 URLs
- Internal linking: every page reaches every other page within 3 clicks

## Architecture

Static HTML generated by `build.py` (single Python 3 file, no dependencies).

To rebuild after editing content:

```bash
python build.py
```

Output is written to the repo root so GitHub Pages serves directly from `main`.
